## 🎯 使用指南

### 快速开始 (5分钟)

```bash
# 1. 克隆项目
git clone https://github.com/AdaMartin18010/OPA.git
cd OPA

# 2. 运行示例
cd examples/01-hello-world
opa test . -v

# 3. 查看文档
# 打开 docs/00-总览与索引.md
```
