## 🚀 快速开始

### 一键启动

```bash
# 1. 克隆项目
git clone https://github.com/AdaMartin18010/OPA.git
cd OPA

# 2. 初始化 (一键完成所有设置)
make setup

# 3. 验证项目完整性
make verify

# 4. 运行测试
make test

# 5. 启动开发服务器
make dev
```

